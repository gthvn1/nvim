# Configuration

- Configuration is done using [lazy.vim](https://github.com/folke/lazy.nvim).

- If you need to backup an old config you can do:
```sh
mv ~/.config/nvim{,.bak}

# optional but recommended
mv ~/.local/share/nvim{,.bak}
mv ~/.local/state/nvim{,.bak}
mv ~/.cache/nvim{,.bak}
```
